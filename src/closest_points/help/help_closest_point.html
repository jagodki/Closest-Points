This algorithm iterates over all features of an input layer and calculates the closest point for each feature to a second layer. Only one point per input feature will be calculated.

Layer to calculate closest points from:

This combobox shows all vector layers of the current QGIS project. For each feature of this layer the closest point will be calculated.

Layer to calculate closest points to:

This combobox shows all vector layers of the current QGIS project. The calculated closest points will be located on features of this layer.

Output:

The destination where the result should be saved.
